Table summary

Name Meaning Description
thead table head thead element defines a set of rows defining the head of the columns of the table. more at MDN
th Table header element element defines a cell as header of a group of table cells. The exact nature of this group is defined by the scope and headers attributes. more at MDN
tbody table body element encapsulates a set of table rows elements, indicating that they comprise the body of the table. more at MDN
tr table row element element defines a row of cells in a table. The row's cells can then be established using a mix of td (data cell) and th (header cell) elements. MDN
td table data element defines a cell of a table that contains data. It participates in the table model. more at MDN
tfoot table foot element defines a set of rows summarizing the columns of the table. more at MDN
attributes Description
colspan Defines how many columns a td element should span.
rowspan Defines how many rows a td element should span.
